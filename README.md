
# CRESSET Lab - Creative Research in Security and Software Engineering Technology

**CRESSET Lab** is a research laboratory at Toronto Metropolitan University led by **Dr. Manar Alalfi**. We focus on cutting-edge research at the intersection of Software Engineering and Cybersecurity, with special emphasis on IoT, Blockchain, Automotive Systems, and AI-powered security analysis.
